# Mafia Mod Installer

Универсальный установщик модов для **Mafia: The City of Lost Heaven**

**Текущая версия: v0.8-beta**

## Основные возможности

- Один исполняемый файл ≈ **12 МБ** (полностью портативный, без установки)
- Поддержка архивов: **ZIP, 7z, RAR, tar, gz** (через встроенные средства и patoolib)
- Drag & Drop для быстрого выбора мода
- Автоматическое определение корневой папки мода
- Создание и восстановление резервных копий игры
- **Hard Restore** — полная замена содержимого папки игры из бэкапа
- Очистка распакованных игровых ресурсов (DTA-файлы)
- **Запуск игры** прямо из программы (Game.exe)
- Умное распознавание readme-файлов на разных языках
- Список установленных модов с датой, количеством файлов и быстрым доступом к readme / бэкапам
- Многоязычный интерфейс: русский · английский · чешский
- Контекстные меню для удобной работы с логом и модами

## Что нового в v0.8-beta

- Сборка в **один .exe-файл** через PyInstaller (~12 МБ)
- Новая кнопка **«Запустить игру»**
- Двойной клик по моду → открывает первый readme-файл
- Контекстное меню по правой кнопке на моду → открыть папку мода / бэкап / readme
- Контекстные меню в логе и вкладке «О программе» (копировать / выделить всё)
- Текст во вкладке «О программе» теперь можно копировать
- Значительно расширен список ключевых слов для поиска readme (многоязычный)
- Исправлено дублирование вкладки «О программе»
- Улучшена стабильность и обработка ошибок

**Рекомендация перед первым использованием:**  
создайте полную резервную копию чистой игры кнопкой «Создать бэкап»
